Types of root devices for the instance
Two types are supported:
- ebs: uses permanent block storage to store data
- instance-store: uses ephemeral block storage to store data

## Values

• **EBS** = ``"ebs"``

___

• **INSTANCE\_STORE** = ``"instance-store"``
